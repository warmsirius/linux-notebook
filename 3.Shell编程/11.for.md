# 11. for

## 11.1 for 语法
```
for 变量 in 值1 值2 值3 ...
do
    程序
done
```

## 11.2 for 案例
1.打印命令行输入的参数 $*

```
#!/bin/bash
for i in "$*"
do
        echo "the num is $i"
done
```

使用$*，输出如下:

```
[root@www shell]# ./fore.sh 10 20 30
the num is 10 20 30
```

```
#!/bin/bash
for i in "$@"
do
        echo "the num is $i"
done
```

使用$@，输出如下:
```
[root@www shell]# ./fore.sh 10 20 30
the num is 10
the num is 20
the num is 30
```

# 20. history指令

## 20.1 history指令语法
history: 查看已经执行过的历史命令，也可以执行历史命令。

功能描述: 查看已经执行过历史命令。

1.查看所有的历史指令
```
history
```

2.查看最近n个历史指令
```
history n
```

3.执行history中编号为n的指令
```
!n
```

3.查看

## 20.2 history指令例子
1.显示所有的历史指令
```
history
```

2.显示最近使用过的10个指令

```
history 10
```

3.执行历史编号为5的指令
```
!5
```

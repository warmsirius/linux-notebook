# 13. more指令

## 13.1 more指令语法
more: 基于VI编辑器的文本过滤器，以全屏幕的方式按页显示文本文件的内容。

```shell script
more 要查看的文件
```

### more操作说明

more指令中内置了若干快捷键，操作说明如下:

| 操作 | 功能说明 |
|---- |---- |
| 空白键[space] | 向下翻一页 |
| Enter | 向下翻一行 |
| q | 立刻离开more，不再显示该文本内容 |
| Ctrl+F | 向下滚动一屏 |
| Ctrl+B | 返回上一屏 |
| = | 输出当前行的行号 |
| :f | 输出文件名和当前行的行号 |

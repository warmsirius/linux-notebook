# 1. date指令

## 1.1 date-显示当前日期
1.显示当前时间
```
date
```

2.显示当前年份
```
date +%Y
```

3.显示当前月份
```
date +%m
```

4.显示当前是哪一天
```
date +%d
```

5.显示当前年月日时分秒

```
date "+%Y-%m-%d %H:%M:%S"
```

## 1.2 date-设置日期

1.设置时间
```
date -s 字符串时间
```

## 1.3 date指令例子
1.设置系统当前时间，设置为2020-11-11 00:00:00

```
date -s "2020-11-11 00:00:00"
```
